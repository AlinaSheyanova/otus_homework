Detailed Description

Represents an action to open a document in the vector editor.

This action sets the active document in the vector editor model based on the specified file path. If the file path is not empty, a new VectorDocument is created with the file path and set as the active document. If the file path is empty, the active document is set to nullptr, indicating no active document.
