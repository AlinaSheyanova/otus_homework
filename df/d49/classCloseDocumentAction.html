Detailed Description

Represents an action to close the active document in a vector editor model.

This class inherits from the IUserAction interface and provides implementation for the Execute and Undo methods.

void CloseDocumentAction::Execute ( )
overridevirtual

Executes the action to close the active document in the vector editor model.

This method gets the current active document from the model, retrieves its file path, sets the active document to nullptr, and stores the file path in the _currentFilePath member variable. It does not return a value.

Note
If there is no active document, the _currentFilePath member variable will remain unchanged.

void CloseDocumentAction::Undo ( )
overridevirtual

Undo the action of closing the active document.

This method checks if the _currentFilePath member variable is not empty. If it is not empty, it creates a new VectorDocument object with the file path stored in _currentFilePath and sets it as the active document in the model. This method does not return a value.

Note
If the _currentFilePath member variable is empty, no action is performed.
