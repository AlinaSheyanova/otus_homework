Detailed Description

A class that represents the action of saving a document to a file.

This class inherits from IUserAction and provides the necessary functionality to execute the save action. It takes a shared_ptr to a VectorEditorModel and a string_view as parameters for the constructor. The execute method saves the active document of the model to the specified file path. The undo method does nothing as there is no action to be undone in this case.
